# Exploratory Sales Analysis

## Description
This project performs exploratory data analysis (EDA) on retail sales data to understand trends, seasonality, and sales distribution.

## Dataset
Store Sales Time Series Forecasting dataset from Kaggle.

## Tasks Performed
- Loaded and cleaned the dataset using Pandas
- Converted date column to datetime format
- Checked missing values and removed duplicates
- Calculated descriptive statistics (mean, median, mode)
- Visualized sales trends using line charts
- Analyzed monthly seasonality using bar charts
- Performed store-wise and product-family-wise sales analysis

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Output
- Visual graphs showing sales trends and seasonality
- Excel file containing summary statistics and monthly sales data
